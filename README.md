## Overview

**CognitiveX** is the official hackathon website designed for our college’s tech fest.  
It provides participants with all essential details such as schedules, registration links, and event information through an interactive and responsive user interface.  
The website is built with a strong focus on creativity, accessibility, and smooth user experience.

---

## Features

- Responsive design that works on mobile, tablet, and desktop  
- Live countdown timer for the hackathon start  
- Announcements section for latest updates  
- Fast performance with optimized assets  
- Clean and modern user interface
